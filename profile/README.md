# Better Call Drone!
Welcome to the **Better Call Drone!** organization. This organization contains the repositories of the project is being developed as part of the Advanced Programming course for the AY 2024/2025 at the University of Trento.

At a high level, the project consists in developing drones, which are deployed in networks, and client-server applications that use the drone network to communicate. All components of the project will be developed in Rust.
